Installing a new operating system

  1. Insert the DVD into the DVD drive
  2. Choose Run
  3. Run the setup.exe
  4. Follow the instructions given by the installer
  5. Reboot your computer after all the files are installed
  6. Cross your fingers and hope your computer doesn't catch fire.
